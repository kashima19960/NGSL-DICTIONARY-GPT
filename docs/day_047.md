### 记忆辅助

## select
单词元数据  
Lemma：select  
SFI Rank：1173  
SFI：58.76  
Adjusted Frequency per Million (U)：75

### 分析词义
- `select`：选择；挑选；精选的。

### 列举例句
- Select one deployment strategy.（选择一种发布策略。）
- We selected candidates by skills.（我们按技能筛选候选人。）
- Please select your region.（请选择你的地区。）

### 词根分析
- se-（分开）+ lect（选择）→挑出。

### 词缀分析
- selection, selective。

### 发展历史和文化背景
- 决策质量很大程度取决于 select 标准是否清晰。

- select/selects/selected/selecting。

### 记忆辅助
- select = “从一堆里挑最合适的”。

### 小故事

## extent

Lemma：extent  
SFI Rank：1175

### 分析词义
- `extent`：程度；范围。

### 列举例句
- To what extent is data delayed?（数据延迟到什么程度？）

### 发展历史和文化背景

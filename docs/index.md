# NGSL Dictionary GPT

欢迎来到 **NGSL 高频英语词汇学习文档**。

本项目基于 `ngsl_src/day_001.csv ~ day_113.csv`，对应生成并维护 `docs/day_001.md ~ day_113.md` 学习内容。

## 使用方式

- 通过左侧导航按天学习词汇；
- 每个单词均包含：词义、例句、词根词缀、词形变化、记忆辅助与小故事；
- 建议按 `day_001 → day_113` 顺序持续学习与复习。
